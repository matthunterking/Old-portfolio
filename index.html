Destination Love

Group Project - 1 week
React | Google Maps API | Filestack | Express | MongoDB | Mongoose | Mocha | Chai | nyc | Heroku | Github | HTML5 | SCSS | JavaScript (ES6)

The brief for this project was to create a full stack web application using React with the option to either work individually or as part of a team. I decided to work in group as I felt that we would be able to produce more than if we worked on our own and working with someone else would provide more opportunities for me to learn. We created a dating site called Destination Love which as well as being a traditional dating site would also solve the dilemma of deciding where to go on a date once you had matched with a potential partner. I primarily worked on the process for users to register, match and message each other as well as the styling. Initially I’d planned to use a third party package to manage the process of sending and accepting match requests however after running into difficulties decided it would be more fun and challenging to create my own system to manage theses requests.
